Prompt Writing Tips

Be Specific

Instead of "Write about AI," try "Write a 500-word blog post about how small businesses can use Claude to automate customer support."

Set a Role

Start with "Act as a senior data analyst" to set Claude's expertise level and response style.

Define Output

Tell the AI exactly what format you want: "Respond in a table with columns for Name, Cost, and Rating."

Give Context

Provide background info: "I am building a React app for a dentist office. The stack is Next.js + Firebase."
