# Cheat Sheet: When to Use Which Tool?

Use this quick-reference guide to instantly know which tool to reach for. Bookmark this page.

## The Decision Table

| Task | Use This | Why |
| :--- | :--- | :--- |
| **Breaking news or current events** | **Perplexity** | Searches the live web in real-time with source URLs |
| **Writing a blog post, essay, or report** | **Claude** | Superior long-form writing with consistent tone |
| **Debugging complex code** | **Claude** | Deep reasoning over large codebases (200k tokens) |
| **Fact-checking a specific claim** | **Perplexity** | Cross-validates via Model Council (3 AI models) |
| **Summarizing a PDF or document** | **Claude** | Upload to Projects for persistent context |
| **Finding the best product to buy** | **Perplexity** | Real-time pricing, reviews, and local availability |
| **Building a website or app** | **Claude** | Claude Code writes, tests, and deploys code |
| **Organizing research into an article** | **Perplexity** | Pages converts search threads into published articles |
| **Automating a multi-step workflow** | **Claude** | Workflows + MCP for end-to-end automation |
| **Getting a second opinion on AI output** | **Perplexity** | Model Council runs GPT, Gemini, and Claude in parallel |
| **Managing local files on your computer** | **Claude** | MCP + Computer Use for local OS operations |
| **Tracking personal finances** | **Perplexity** | Personal CFO with Plaid bank integration |

## The Golden Rule

> **Expert Tip:** If the answer requires **today's information from the internet**, use Perplexity. If the answer requires **deep thinking, creativity, or working with your private files**, use Claude. If you need both, use the **AI Sandwich** method (covered in the Workflows section).

## Quick Decision Flowchart

Ask yourself these 3 questions:

1. **Does it need live internet data?** → Yes → **Perplexity**
2. **Does it need to analyze my private files?** → Yes → **Claude Projects**
3. **Does it need creative writing or code?** → Yes → **Claude**
4. **Not sure?** → Start with Perplexity to research, then feed the results to Claude to create.

## Pro Tips for 2026

- **Claude:** Use the `xhigh` effort level for complex tasks. Use `/ultrareview` in Claude Code to audit your code before committing.
- **Perplexity:** Use "Focus" modes to narrow your search (Web, Academic, Reddit, YouTube). Use Model Council for high-stakes decisions.
- **Both:** Always set up your "AI Profile" or "Custom Instructions" so each tool knows your role, industry, and preferences.
