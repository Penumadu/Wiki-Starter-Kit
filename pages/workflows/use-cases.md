# Real-World Use Case Samples

**Summary:** Learn how to combine Claude and Perplexity to solve complex, real-world problems in minutes rather than hours.

---

### 💻 Coding & Development
**The Scenario:** You need to build a React application that uses a brand-new API with limited documentation.
- **Perplexity's Role:** Use **Pro Search** to find the latest API endpoints, authentication methods, and community forum discussions.
- **Claude's Role:** Paste the research into a **Project**. Use **Claude Code** to scaffold the React app and **Artifacts** to preview the UI in real-time.
- **The "Aha!" Moment:** Realizing Claude fixed a bug in the code because Perplexity found a GitHub issue comment from 2 hours ago that you hadn't seen.

---

### 🏢 Real Estate & Business
**The Scenario:** You need to research zoning laws in a specific city and draft a professional proposal for a new development.
- **Perplexity's Role:** Use **Computer Agent** to browse the city's official zoning portal and download the latest PDF bylaws.
- **Claude's Role:** Upload the bylaws to a **Project**. Ask Claude to "Analyze these laws for height restrictions" and draft a formal proposal in your company's tone.
- **The "Aha!" Moment:** Having a 10-page proposal drafted and cited in under 15 minutes.

---

### 🎓 Personal & Education
**The Scenario:** You are studying for a complex certification (e.g., AI-102) and need a study guide and practice quiz.
- **Perplexity's Role:** Use **Pages** to create a comprehensive study guide by searching all the official Microsoft documentation and community blogs.
- **Claude's Role:** Copy the link to your Perplexity Page into Claude. Ask Claude to "Quiz me on these 5 topics" and provide feedback on your answers.
- **The "Aha!" Moment:** Realizing you just created a custom, interactive tutor that knows the exact latest exam updates.
